## Features

### Authentication & Authorization
- JWT-based authentication
- Secure password hashing (bcrypt)
- Role-based access control (User / Admin)

### Problem Management
- Create & manage coding problems
- Difficulty levels: `Easy | Medium | Hard`
- Tags & constraints
- Sample and hidden test cases
- Unique slug-based problem access

### Competitive Programming
- Weekly / custom contests (planned)
- User rating system
- Solved problem tracking
- Leaderboard (planned)

### User System
- User profiles
- Solved count & rating
- Admin-only routes for moderation
